# Personal Website

Built using:

-   TypeScript (type checking)
-   React (Component-based UI framework)
-   Vite (build tool)
-   Material UI (styling).

## Structure

```
website/
├── index.html
└── src/

    │       ├── ContactPage.tsx
    │       ├── ProjectsPage.tsx
    │       ├── ToolsPage.tsx
    │       └── index.ts
    ├── main.tsx
    └── vite-env.d.ts
```
